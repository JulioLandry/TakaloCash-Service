Structure des dossiers d'assets

Créez cette structure de dossiers dans votre projet :

takalocash/
├── assets/
│   ├── icons/
│   │   ├── icon-72x72.png
│   │   ├── icon-96x96.png
│   │   ├── icon-128x128.png
│   │   ├── icon-144x144.png
│   │   ├── icon-152x152.png
│   │   ├── icon-192x192.png
│   │   ├── icon-384x384.png
│   │   └── icon-512x512.png
│   ├── images/
│   │   ├── logo.png
│   │   ├── logo.svg
│   │   └── banner.jpg
│   └── favicon/
│       └── favicon.ico

Code pour intégration

Ajoutez ce code dans le <head> de votre index.html :

<!-- Favicon -->
<link rel="icon" type="image/x-icon" href="assets/favicon/favicon.ico">
<link rel="apple-touch-icon" href="assets/icons/icon-192x192.png">

<!-- Manifest PWA -->
<link rel="manifest" href="manifest.json">
